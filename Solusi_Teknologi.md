| No. | Solusi Teknologi | Deskripsi Singkat | Tools/OSS yang Digunakan |
|-----|------------------|-------------------|--------------------------|
| 1 | Platform Tata Kelola Data untuk Kebijakan Pangan Nasional | Mengembangkan sistem terpusat untuk mengelola metadata dan kebijakan data terkait program makan bergizi gratis, memastikan kepatuhan PESTLE melalui kontrol akses dan audit trail, dengan fokus pada aspek people melalui user-friendly interface. | OpenMetadata (untuk katalogisasi data) dan Apache Atlas (untuk governance metadata). |
| 2 | Arsitektur Data Lakehouse untuk Integrasi Data Gizi | Membangun repositori data terintegrasi yang menggabungkan data mentah dan terstruktur dari berbagai sumber keamanan pangan, mendukung analisis prediktif untuk ketahanan pangan dengan skalabilitas tinggi. | Apache Iceberg (untuk table format ACID) dan Delta Lake (untuk transaksi data lake). |
| 3 | Sistem Analisis Sentimen Berbasis NLP untuk Feedback Program | Menganalisis opini masyarakat terhadap program makan bergizi gratis dari media sosial dan survei, menggunakan NLP untuk mendeteksi tren sosial dan mendukung aspek process dalam PPT. | Hugging Face Transformers (untuk model pre-trained) dan spaCy (untuk pemrosesan teks cepat). |
| 4 | Pipeline Otomatis Pengelolaan Kualitas Data Rantai Pasok | Mengimplementasikan pipeline untuk memvalidasi dan membersihkan data kualitas pangan secara real-time, mengurangi risiko kesalahan dalam distribusi program gratis dengan metrik otomatis. | Great Expectations (untuk testing data) dan Deequ (untuk profiling kualitas di Spark). |
| 5 | Dashboard Prediktif Ketahanan Pangan dengan Lakehouse | Solusi visualisasi data untuk memprediksi kekurangan gizi berdasarkan faktor PESTLE, menggunakan lakehouse untuk menyimpan data historis dan teknologi ML untuk forecasting. | Apache Pinot (untuk OLAP real-time) dan Polars (untuk manipulasi data efisien). |
| 6 | Framework Governance dengan Kontrol Akses Berbasis Role | Mengembangkan sistem keamanan data untuk melindungi informasi sensitif keamanan pangan, terintegrasi dengan autentikasi untuk mendukung aspek technology dalam PPT di Badan Gizi Nasional. | Apache Ranger (untuk authorization) dan Keycloak (untuk manajemen identitas). |
| 7 | Alat Ekstraksi Insight NLP dari Laporan Keamanan Pangan | Menggunakan NLP untuk mengekstrak entitas dan tema dari dokumen regulasi dan laporan, membantu analisis legal dan environmental dalam PESTLE untuk program bergizi. | NLTK (untuk tokenisasi dasar) dan Gensim (untuk topic modeling). |
| 8 | Sistem Monitoring Kualitas Data Berbasis AI untuk Program Gratis | Otomatisasi deteksi anomali data distribusi makanan bergizi, dengan integrasi AI untuk rekomendasi perbaikan, fokus pada process improvement dalam PPT. | Soda Core (untuk observability) dan DQOps (untuk testing sensor-based). |
| 9 | Integrasi Lakehouse dengan Governance untuk Data Lingkungan | Solusi hybrid untuk menyatukan data iklim dan pangan dalam lakehouse, dengan layer governance untuk traceability, mendukung analisis environmental PESTLE. | Apache Arrow (untuk in-memory processing) dan Amundsen (untuk discovery data). |
| 10 | Platform NLP Terintegrasi DQM untuk Data Teks Sosial | Menggabungkan NLP dengan DQM untuk membersihkan dan menganalisis data teks dari survei gizi, memastikan kualitas input untuk insight akurat terkait ketahanan pangan. | TextBlob (untuk sentiment sederhana) dan dbt Core (untuk transformasi data quality). |
